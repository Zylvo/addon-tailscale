# Home Assistant Add-ons

This repository contains my add-ons for Home assistant.

## Installation

Navigate to **Settings** -> **Add-ons** -> **Add-on Store** and add this URL as an additional repository:
```txt
https://github.com/Zylvo/zylvo-haaddons
```
